# Exploratory Data Analysis

This project focuses on exploratory data analysis of movies and TV shows data from a streaming platform.

The objective of this project is to understand content performance, audience preferences, and trends using data analysis and visualization techniques.

## Tools & Libraries Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## Key Analysis Performed
- Data cleaning and preprocessing  
- Univariate, bivariate, and multivariate analysis  
- Genre, rating, popularity, and production country analysis  
- Trend analysis over years  
- Correlation and pair plot analysis  

## Outcome
The analysis provides useful insights that can help improve content planning, recommendation strategy, and business decision making.
